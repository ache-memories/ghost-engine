# 👻 SEOGhostEngine

A serverless, high-speed **SEO Metadata Scraper** built using clean JavaScript and HTML. Instantly audit any URL to extract Meta Titles, Descriptions, and H1 tags without the browser blocking CORS.

## ✨ Features

- 🚀 **Fast Extraction:** Grabs Title, Description, and H1 in milliseconds.
- 🛡️ **CORS Bypass:** Uses a built-in proxy to fetch data from any site.
- 💻 **Serverless:** Written in pure HTML/JS, it runs directly in your browser.
- 🎨 **Modern Dark UI:** Beautiful, responsive interface for specialists and developers.
